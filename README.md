# 🎥 Smart Motion Surveillance System (SMSS)

## 📌 Description
The **Smart Motion Surveillance System (SMSS)** is a Python-based desktop application designed to provide real-time motion detection and automated video recording. Built using **OpenCV** and **PyQt5**, the system monitors a live camera feed, detects motion within a defined region, and records both full-frame and focused-area videos when activity is detected.

The application features an interactive GUI dashboard, allowing users to control surveillance operations, adjust motion sensitivity dynamically, and receive real-time status updates. It also includes a secure login system to restrict unauthorized access.

This project demonstrates the integration of **computer vision**, **GUI development**, and **multithreading** to build a practical, real-world surveillance solution.

---

## 🚀 Key Features
- 🎯 Real-time Motion Detection using OpenCV  
- 🎥 Automatic Video Recording (Full Frame + ROI)  
- 🖥 Interactive GUI built with PyQt5  
- 🔐 Secure Login System with limited attempts  
- 🎚 Dynamic Motion Sensitivity Control (Slider-based)  
- 🔊 Sound Alerts on Motion Detection  
- 📊 Live Status Updates (Idle / Running / Recording)  
- ⏱ Timestamp and Recording Duration Overlay
